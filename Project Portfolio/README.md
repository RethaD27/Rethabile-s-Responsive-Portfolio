# DevRethabile Portfolio Website

This is a portfolio website for Rethabile Diale, showcasing his skills and projects.

## Description

This website serves as a platform for Rethabile Diale to present his work as a front-end developer and math tutor. It includes sections such as an introduction, services offered, information about Rethabile, and a portfolio of his work.

## Usage

To view the website, simply open the `index.html` file in a web browser. You can navigate through the different sections using the navigation links in the header.

## Sections

- **Home**: Introduction to Rethabile Diale.
- **My Services**: Information about the services offered by Rethabile, including front-end web development and math tutoring.
- **About Me**: Details about Rethabile's background and interests.
- **My Work**: Portfolio showcasing Rethabile's range of work.
